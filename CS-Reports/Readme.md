# CS Labs — Computer Science / Cybersecurity

This folder contains all Computer Science / Cybersecurity lab materials, organized into original college labs and professional-style reworked reports.

## Purpose

- Document foundational and advanced CS/cybersecurity labs in a structured and professional way.  
- Showcase learning progression from academic exercises to portfolio-ready reports.  
- Maintain legal, isolated lab work for reference and reproducibility.

## Notes

- Labs cover topics including Kali Linux, Bash scripting, Wireshark analysis, ARP poisoning, Metasploitable exploitation, Snort, Fail2ban, Hydra, and Nmap.  
- All work performed in controlled virtual lab environments.  
- Polished reports are text-based and professional.
